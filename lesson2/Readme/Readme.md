## Урок 2. Неймспейсы и изоляция

Контейнеризация, как концепция, и технологии изоляции, используемые для реализации контейнеров, имеют глубокие корни в мире Linux и технологиях ядра операционной системы.

Идея контейнеризации начала своё развитие в Unix-подобных системах ещё в конце прошлого века, когда были введены концепции виртуальной машины (chroot) и пространства пользователя (user namespaces). Они предоставляли базовую изоляцию процессов и файловой системы, позволяя создавать относительно изолированные окружения.

Однако, реализация контейнеризации в полной мере стала возможной лишь с развитием технологий ядра Linux, таких как control groups (cgroups) и namespaces. Эти функции стали стандартом для создания изолированных окружений и управления ресурсами.
